# Divya Portfolio (React + Vite)

A modern, colorful, animation-rich personal portfolio website built with React and Vite.

## Highlights
- Premium dark UI (`#020617` base)
- Multi-color gradient accents and glow effects
- Glassmorphism components
- Hero typing animation
- Scroll reveal animations with staggered timing
- Animated skills progress bars
- Fully responsive layout (desktop + mobile)

## Project Structure
- `index.html` - Vite app shell
- `src/main.jsx` - React entry point
- `src/App.jsx` - Main UI and interactions
- `src/styles.css` - Full visual system and animation styles
- `DOCS.md` - Extended project documentation

## Getting Started
### 1) Install dependencies
```bash
npm install
```

### 2) Run development server
```bash
npm run dev -- --host
```

### 3) Build production bundle
```bash
npm run build
```

### 4) Preview production build
```bash
npm run preview
```

## NPM Scripts
- `npm run dev` - Start Vite dev server
- `npm run build` - Build production assets into `dist/`
- `npm run preview` - Preview production build locally

## Deployment
Deploy the `dist/` folder to any static host:
- Vercel
- Netlify
- GitHub Pages
- Firebase Hosting

## Notes
- Current contact form has front-end-only interaction.
